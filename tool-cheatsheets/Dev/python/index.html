Print raw bytes

Python2:

$ python2 -c 'print("\xff")' | hexdump -C
00000000  ff 0a                                             |..|
00000002

$ python2 -c 'import sys;sys.stdout.write("\xff")' | hexdump -C
00000000  ff                                                |.|
00000001

Python3:

$ python3 -c 'import sys;sys.stdout.buffer.write(b"\xff")' | hexdump -C
00000000  ff                                                |.|
00000001

struct

This module performs conversions between Python values and C structs represented as Python bytes objects. This can be used in handling binary data stored in files or from network connections, among other sources. It uses Format Strings as compact descriptions of the layout of the C structs and the intended conversion to/from Python values. 1

import struct

# pack into a x64 little endian string
struct.pack("<Q",0x1234)
b'4\x12\x00\x00\x00\x00\x00\x00'

# pack into a x64 big endian string
struct.pack(">Q",1234)
b'\x00\x00\x00\x00\x00\x00\x04\xd2'

# pack into a x86 little endian string
struct.pack("<I",0x1234)
b'4\x12\x00\x00'

# pack into a x86 big endian string
struct.pack(">I",0x1234)
b'\x00\x00\x124'

# unpack from a x86 big endian string
struct.unpack(">I",b'\x00\x00\x124')[0]
4660
hex(struct.unpack(">I",b'\x00\x00\x12\x34')[0])
'0x1234'
